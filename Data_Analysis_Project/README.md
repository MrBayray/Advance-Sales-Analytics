# 📊 Monthly Demand Trend Analysis

## 📌 Overview
This project analyzes monthly passenger/sales data to identify trends and seasonal demand patterns.

## 🧰 Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Key Steps
- Data loading and cleaning
- Grouping data by month
- Calculating average demand
- Visualization using bar chart

## 📈 Key Insight
Some months show higher demand than others, indicating seasonal variation in activity.

## 🚀 Goal
To practice real-world data analysis and build a portfolio-ready project for junior data analyst roles.
